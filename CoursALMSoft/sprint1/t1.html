Task1 -- Flot d'exécution d'instructions élémentaires

Cette première tâche concerne le flot d'exécution, c'est-à-dire la façon dont le processeur exécute les instructions d'un programme en séquence, sans jamais s'arrêter. Dans un terminal de commande placez vous dans le répertoire sprint1 / task1, où vous trouverez tout ce dont vous avez besoin:

  1. Le programme source
  2. Un fichier Makefile
  3. Un fichier .gdbinit

A noter qu'il faudra procéder de la même façon pour les tâches suivantes, chacune étant associée à un répertoire de même nom.

    4. Le même programme écrit dans le langage C
